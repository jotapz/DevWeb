Questão 1

O que é o JavaScript?

Resposta:

JavaScript é uma linguagem de programação leve, interpretada e orientada a objetos, amplamente utilizada para criar páginas web interativas. Ela permite manipular o conteúdo HTML e CSS, responder a eventos do usuário, validar formulários, criar animações, entre outras funcionalidades. JavaScript é executado no lado do cliente (navegador) e também pode ser usado no lado do servidor com ambientes como Node.js.

Questão 2

Qual a diferença entre var, let e const?

Resposta:

As principais diferenças entre var, let e const são:


  • var: Tem escopo de função, ou seja, é visível em toda a função onde foi declarada. Pode ser redeclarada e atualizada. Sofre hoisting, o que significa que sua declaração é "elevada" ao topo do escopo.

  • let: Tem escopo de bloco, ou seja, é visível apenas dentro do bloco onde foi declarada (entre chaves {}). Pode ser atualizada, mas não redeclarada no mesmo escopo. Também sofre hoisting, mas não pode ser acessada antes da declaração (tem um "temporal dead zone").

  • const: Também tem escopo de bloco. Não pode ser redeclarada nem atualizada (o valor deve ser atribuído na declaração). No entanto, se o valor for um objeto ou array, suas propriedades ou elementos podem ser modificados. Sofre hoisting similar ao let.
Questão 3

O que são funções em JavaScript?

Resposta:

Funções em JavaScript são blocos de código reutilizáveis que executam uma tarefa específica. Elas podem receber parâmetros de entrada, realizar operações e retornar um valor. As funções ajudam a organizar o código, tornando-o mais modular e fácil de manter.

Questão 4

O que é o DOM (Document Object Model)?

Resposta:

O DOM (Document Object Model) é uma interface de programação para documentos HTML e XML. Ele representa a estrutura da página como uma árvore de objetos, onde cada nó é um elemento, atributo ou texto do documento. O DOM permite que linguagens de programação, como JavaScript, acessem e manipulem o conteúdo, estrutura e estilo da página web dinamicamente.

Questão 5

Como funciona a execução síncrona e assíncrona no JavaScript?

Resposta:

JavaScript é uma linguagem de programação single-threaded, o que significa que ele executa um comando por vez em uma única linha do tempo. A execução síncrona ocorre quando o código é executado linha por linha, bloqueando a execução até que a tarefa atual seja concluída. Isso pode levar a problemas de desempenho se uma tarefa demorar muito tempo para ser concluída, como uma requisição de rede ou uma operação de leitura/escrita em disco.

Questão 6

O que são Promises e como elas são usadas?

Resposta:

Promises são objetos que representam a eventual conclusão (ou falha) de uma operação assíncrona e seu valor resultante. Elas permitem que você escreva código assíncrono de forma mais legível e gerenciável, evitando o "callback hell".

Questão 7

Explique o uso de async e await em JavaScript

Resposta:

async e await são palavras-chave introduzidas no ES2017 que facilitam o trabalho com código assíncrono em JavaScript. A palavra-chave async é usada para declarar uma função assíncrona, que sempre retorna uma Promise. Dentro de uma função async, você pode usar a palavra-chave await antes de uma expressão que retorna uma Promise, fazendo com que a execução da função pause até que a Promise seja resolvida ou rejeitada. Isso permite escrever código assíncrono de forma mais linear e fácil de entender, semelhante ao código síncrono.

Questão 8

Qual a diferença entre for...in e for...of?

Resposta:

A principal diferença entre for...in e for...of em JavaScript está no tipo de dados que cada um itera e como eles funcionam:


  • for...in: Itera sobre as propriedades enumeráveis de um objeto. Ele percorre as chaves (nomes das propriedades) do objeto, incluindo propriedades herdadas da cadeia de protótipos. É mais adequado para objetos, mas pode ser usado em arrays, embora não seja recomendado, pois a ordem das propriedades não é garantida.

  • for...of: Itera sobre valores iteráveis, como arrays, strings, mapas, conjuntos e outros objetos que implementam o protocolo iterável. Ele percorre os valores diretamente, mantendo a ordem dos elementos. É mais adequado para arrays e outros tipos de dados iteráveis.
Questão 9

O que é desestruturação (destructuring) em JavaScript?

Resposta:

Desestruturação é uma sintaxe em JavaScript que permite extrair valores de arrays ou propriedades de objetos e atribuí-los a variáveis de forma mais concisa e legível. Ela facilita o acesso a dados complexos, reduzindo a necessidade de múltiplas linhas de código para acessar cada valor individualmente.

Questão 10

O que são os operadores spread e rest em JavaScript?

Resposta:

Os operadores spread (...) e rest (...) são dois usos diferentes do mesmo símbolo em JavaScript, mas com propósitos distintos:


  • Operador Spread: O operador spread é usado para "espalhar" os elementos de um array ou as propriedades de um objeto em outro array ou objeto. Ele é útil para copiar arrays, combinar arrays, adicionar elementos a um array existente ou clonar objetos.

  • Operador Rest: O operador rest é usado para agrupar múltiplos elementos em um único array. Ele é frequentemente utilizado em funções para capturar um número variável de argumentos passados para a função, permitindo que você trabalhe com esses argumentos como um array.

Questão 17

Requisição fetch() - API externa

📝 Enunciado

Desenvolva um código que faça uma requisição a uma API pública, como JSONPlaceholder, e mostre o nome de um usuário na página.

💡 Resposta

Clique no botão para buscar usuário
// JavaScript - Versão com async/await
async function buscarUsuario() {
  const userDiv = document.getElementById('usuario-info');
  
  try {
    userDiv.innerHTML = 'Carregando...';
    
    const response = await fetch('https://jsonplaceholder.typicode.com/users/1');
    
    if (!response.ok) {
      throw new Error('Usuário não encontrado');
    }
    
    const usuario = await response.json();
    
    userDiv.innerHTML = `
      

        
${usuario.name}

        
📧 ${usuario.email}

        
📞 ${usuario.phone}

        
🌐 ${usuario.website}

        
🏢 ${usuario.company.name}

      

    `;
  } catch (error) {
    userDiv.innerHTML = `
❌ ${error.message}
`;
  }
}

// Adicionar evento ao botão
document.getElementById('buscar-usuario').addEventListener('click', buscarUsuario);
Questão 18

LocalStorage - armazenar JSON

📝 Enunciado

Escreva o código para salvar um objeto como uma string JSON no LocalStorage e depois recuperá-lo.

💡 Resposta

// Objeto exemplo
const usuario = {
  id: 1,
  nome: 'João Silva',
  email: 'joao@email.com',
  idade: 30,
  preferencias: {
    tema: 'escuro',
    idioma: 'pt-br'
  },
  interesses: ['programação', 'música', 'esportes']
};

// 1. SALVAR objeto no LocalStorage
function salvarUsuario(dadosUsuario) {
  try {
    // Converter objeto para JSON string
    const usuarioJSON = JSON.stringify(dadosUsuario);
    
    // Salvar no LocalStorage
    localStorage.setItem('usuario', usuarioJSON);
    
    console.log('Usuário salvo com sucesso!');
  } catch (error) {
    console.error('Erro ao salvar usuário:', error);
  }
}

// 2. RECUPERAR objeto do LocalStorage
function recuperarUsuario() {
  try {
    // Buscar JSON string do LocalStorage
    const usuarioJSON = localStorage.getItem('usuario');
    
    // Verificar se existe
    if (usuarioJSON === null) {
      console.log('Nenhum usuário encontrado no LocalStorage');
      return null;
    }
    
    // Converter JSON string de volta para objeto
    const usuario = JSON.parse(usuarioJSON);
    
    console.log('Usuário recuperado:', usuario);
    return usuario;
  } catch (error) {
    console.error('Erro ao recuperar usuário:', error);
    return null;
  }
}

// 3. ATUALIZAR dados no LocalStorage
function atualizarUsuario(novosDados) {
  const usuarioAtual = recuperarUsuario();
  
  if (usuarioAtual) {
    // Mesclar dados existentes com novos dados
    const usuarioAtualizado = { ...usuarioAtual, ...novosDados };
    salvarUsuario(usuarioAtualizado);
  }
}

// 4. REMOVER usuário do LocalStorage
function removerUsuario() {
  localStorage.removeItem('usuario');
  console.log('Usuário removido do LocalStorage');
}

// Exemplos de uso:
salvarUsuario(usuario);
const usuarioRecuperado = recuperarUsuario();
console.log('Nome:', usuarioRecuperado?.nome);
Questão 19

Validação de Formulário

📝 Enunciado

Crie um formulário HTML com um campo de nome e um botão de envio. Use JavaScript para exibir um alerta caso o campo esteja vazio.

💡 Resposta

// JavaScript - Validação básica
document.getElementById('meuFormulario').addEventListener('submit', function(event) {
  event.preventDefault(); // Impede o envio padrão do formulário
  
  const nome = document.getElementById('nome').value.trim();
  const email = document.getElementById('email').value.trim();
  
  // Verificar se o nome está vazio
  if (nome === '') {
    alert('Por favor, preencha o campo Nome!');
    document.getElementById('nome').focus();
    return;
  }
  
  // Verificar se o email está vazio
  if (email === '') {
    alert('Por favor, preencha o campo Email!');
    document.getElementById('email').focus();
    return;
  }
  
  // Se chegou até aqui, formulário é válido
  alert(`Formulário enviado com sucesso!\nNome: ${nome}\nEmail: ${email}`);
});

Relógio Digital

📝 Enunciado

Crie um relógio digital que exiba a hora atual atualizando a cada segundo usando setInterval().

💡 Resposta

00:00:00
Carregando...
// JavaScript - Relógio Digital
let intervaloRelogio;
let relogioPausado = false;
let formato24h = true;

// Função para atualizar o relógio
function atualizarRelogio() {
  const agora = new Date();
  
  let horas = agora.getHours();
  let minutos = agora.getMinutes();
  let segundos = agora.getSeconds();
  let periodo = '';
  
  // Formato 12h ou 24h
  if (!formato24h) {
    periodo = horas >= 12 ? ' PM' : ' AM';
    horas = horas % 12;
    horas = horas ? horas : 12; // 0 vira 12
  }
  
  // Adicionar zero à esquerda se necessário
  horas = horas.toString().padStart(2, '0');
  minutos = minutos.toString().padStart(2, '0');
  segundos = segundos.toString().padStart(2, '0');
  
  // Atualizar display do relógio
  const horaFormatada = `${horas}:${minutos}:${segundos}${periodo}`;
  document.getElementById('relogio').textContent = horaFormatada;
  
  // Atualizar data
  const diasSemana = ['Domingo', 'Segunda', 'Terça', 'Quarta', 'Quinta', 'Sexta', 'Sábado'];
  const meses = ['Janeiro', 'Fevereiro', 'Março', 'Abril', 'Maio', 'Junho',
                 'Julho', 'Agosto', 'Setembro', 'Outubro', 'Novembro', 'Dezembro'];
  
  const diaSemana = diasSemana[agora.getDay()];
  const dia = agora.getDate();
  const mes = meses[agora.getMonth()];
  const ano = agora.getFullYear();
  
  const dataFormatada = `${diaSemana}, ${dia} de ${mes} de ${ano}`;
  document.getElementById('data').textContent = dataFormatada;
}

// Função para iniciar o relógio
function iniciarRelogio() {
  atualizarRelogio(); // Atualizar imediatamente
  intervaloRelogio = setInterval(atualizarRelogio, 1000); // Atualizar a cada segundo
}

// Iniciar o relógio quando a página carregar
iniciarRelogio();
