1. Oque é HTML? Explique o que é HTML e sua importância no desenvolvimento web.

HTML é uma sigla para HyperText Markup Language, é a linguagem fundamental para a criação e estruturação de páginas web, sendo o esqueleto de um site, definindo a hierarquia e organização do conteúdo

A importância do HTML no desenvolvimento web é imensurável, sendo o pilar sobre o qual toda a web é construída. Sem ele, os navegadores não teriam um padrão para interpretar e exibir o conteúdo das páginas
