7. Quais são os principais elementos usados para criar um formulário HTML? Cite pelo menos cinco e explique suas funções.

FORM

Função: É o elemento principal que funciona como um contêiner para todos os outros elementos de entrada de dados. Ele define o início e o fim do formulário.

LABEL

Função: Associa um texto descritivo a um elemento de entrada input, sendo crucial para a acessibilidade, pois permite que leitores de tela identifiquem corretamente cada campo.

INPUT

Função: Este elemento é o mais versatil e fundamennntal de um formulário, cria diversos tipos de campos de entrada de dados dependendo do valor do seu atributo type que podem ser text, password, email e entre outros.

TEXTAREA

Função: É usada para criar um campo de entrada de texto de múltipla linhas, ideal para coletar textos longos, como comentários, mensagens ou descrições.

SELECT

Função: Cria uuma lista de opções suspensas, onde cada item selecionável dentro dessa lista é definido pela tag option. É uma forma eficiente de permitir que o usuário escolha um valor a partirt de uma lista pré-definida de opções.
