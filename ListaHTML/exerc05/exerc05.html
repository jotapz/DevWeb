5. Explique a diferença entre ul, ol e dl. Quando cada uma deve ser usada?

UL: Lista não ordenada

Ela agrupa uma coleção de itens onde a ordem não importa.

Quando usar

  • Lista ingredientes de uma receita, ond a ordem não importad
  • Menus de navegação em sites
  • Listar características
  • Qualquer lista onde a sequência dos itens pode ser alterada sem mudar o sentido

OL: Lista ordenada

Esta lista é usada oara agrupar itens onde a ordem é crucial

Quando usar

  1. Instruções passo a passo de receita ou tutorial
  2. Um ranking de classificação
  3. Etapas de um processo
  4. Qualquer lista onde a posição é fundamental

DL: Lista de definição

Essa lista é diferente, não é para itens simples, mas sim para pares de termo e descriçãod

Quando usar

Termo ou nome chave
Descrição ou definição
