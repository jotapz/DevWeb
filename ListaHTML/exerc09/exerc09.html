9. O que são meta tags e para que servem? Dê dois exemplos de uso.

As MetaTags são elemtnos HTML que descrevem características de uma página web, como o conjunto de caracteres utilizado, a descrição do conteúdo para mecanismos de busca, o autor do docu,mento, palavras-chhave e diretivas para o comportamento do navegador

Exemplos de uso

1. Definição do conjunto de caracteres

MetaTag mais fundamental, ela informa ao navegador qual codificação de caracteres deve ser usada para interpretar o texto da págica, sem ela caracteres especiais como acentos e cedilha poderiam aparecer desconfigurados


2. Descrição da página

Criar um resumo atrativo para que o usuário clique no seu link nos resultados de uma busca. Uma boa descrição pode aumentar significativamente a taxa de cliques.
