4. O que são elementos HTML e atributos? Dê um exemplo de uso de um atributo dentro de uma tag HTML.

Elementos

Um elemennnto em HTML é uma parte individual de uma página web, definido por uma tag de início, pode conter algum conteúdo e geralmente uma tag de fechamento.

Por exemplo para criar um parágrafo utilizamos a TAG p entre <>

Atributos

Os atributos HTML fornecem informações adicionais sobre um elemento.

Eles são sempre especificados na tag de abertura e geralmente vêm em pares de nome/valor, como nome="valor"
