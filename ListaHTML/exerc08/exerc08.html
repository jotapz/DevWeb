8. Qual a diferença entre um link absoluto e um link relativo em HTML? Dê um exemplopara cada caso.

Link Absoluto

Um link absoluto contém o endereço completo e exato de um recurso na internet, inclui o protocolo http:// ou https://, o nome do domínico e opcionalmente o caminho para o arquivo específico.

Exemplo de um link absoluto: Meu GitHub

Outro exemplo: PokemonDB

Link Relativo

Um link relativo aponta para um recurso especificando o caminho em relação à localização do arquivo atual. Ele não inclui o protocolo ou o nome do domínio, pois presume que o recurso está no mesmo servidor.

Exemplo de link relativo: voltar para o enunciado
