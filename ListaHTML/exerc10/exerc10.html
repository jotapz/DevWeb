10. Quais são as tags para incorporar áudio e vídeo em um site? Como garantir compatibilidade com diferentes navegadores?

A tag video é usada para incorporar conteúdo de vídeo, como filmes, clipes ou transmissões

A tag audio é usada para incorporar conteúdo de som em um documento, como música, podscasts e outros fluxos de aúdio

A melhor maneira de garantir máxima compartibilidade é fornecer múltiplos formatos de arquivo para o meesmo conteúdo usando a tag SOURCE

A tag source é colocada dentroo das tags AUDIO e VIDEO, permitindo especificar vários arquivos de mídia alternativos
